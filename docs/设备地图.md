## 四、命名规则

### 设备命名
- 主力设备 → 动物庄园里的【猪】(Napoleon / Snowball / Old Major / Squealer 等)
- 工作设备 → 项目名 (xyc) 或其他动物 (Boxer / Benjamin / Mollie 等)
- 选名理由: 体系化, 有典故, 易记

### Agent 命名
- 标准格式: `<设备名>_<编译器名>`
  - 示例: `OldMajor_Codex`, `XYC_Windsurf`
- 简写: 取代号
  - 示例: `XYC_Windsurf` → `DogWind`
- 同设备多 Agent: `<设备名>_<编译器名>_<序号>`
  - 示例: `OldMajor_Codex_2`

### 任务编号
- `RU-XXX`: RevUnsup 主线 (xyc 5060 上做的事)
- `OM-XXX`: Old Major 副线 (2080 上做的事)
- 主线和副线编号独立, 避免冲突
